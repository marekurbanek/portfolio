PROJECTS

GymBuddy

    After completing over 30 hours of the React course and the ecosystem associated with it, I decided that it is the right time to consolidate knowledge by doing a project. Sometimes I workout in the gym, and I noticed that there is no application that would allow to keep a training diary in a simple and flexible way, so I decided to change that. The application is made using React + Redux.
    Currently, it has basic functionality of training log, i.e. adding/removing workouts, adding/removing exercises within training along with series, repetitions, etc. App isn’t ready and I’m planning to add a server, and database with users who can arrange workouts together with another users.

Wild Camps

    This is the most complex application I have made. I encountered here, among others with such concepts as: non-relational databases, CRUD operations, user authentication, data validation, session, using external API. This application is also related to my interests, which in this case are travels.
    In this app the community can share their camping places with photos, providing info such as: location and description. It is possible to add, edit and delete campings/comments/user profile. After switching to the user profile, his profile picture is shown, among with description, counter of added places and underneath it there is a section with all campsites of the user. Everything is shared using Heroku web servers.

Text Learning

    Simple React app, created to help students memorize long texts. It uses an effective learning method, which depends on hiding certain words in the text, memorizing them and then increasing the number of hidden words until the whole text is memorized.
    To cover the word, click once. To see the word, just move the mouse over it. To mark a word as memorized, click it second time.
