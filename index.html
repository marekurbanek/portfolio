PROJECTS

GymBuddy

    After completing over 30 hours of the React course and the ecosystem associated with it, I decided to consolidate my knowledge by doing a project. Sometimes I work out in the gym, and I noticed that there was no application that would allow a person to keep a training diary in a simple and flexible way, so I decided to change that.
    The application is made using React + Redux. Currently, it has a basic functionality of a training log, i.e. adding/removing workouts, adding/removing exercises within training along with series, repetitions, etc. I am planning to add a server and database to allow users to communicate and arrange workouts with other users. This is the second version of the application, in which I've improved UX, UI and RWD. I've used material UI to make it look good.

Wild Camps

    This is the most complex application I have created. The project has allowed me to learn about non-relational databases, CRUD operations, user authentication, data validation, session, using external API. This application is also related to my interests, which in this case is travelling.
    This app allows the community to share their favorite camping places with photos and provide useful information about the location. It is possible to add, edit and delete camping/comments/user profile. After switching to the user profile, a profile picture is shown, as well as a description, a counter of added places and a section with all campsites of the user. Everything is shared using Heroku web servers.

Text Learning

    Simple React app, created to help students memorize long texts. It uses an effective learning method, which depends on hiding certain words in the text, memorizing them and then increasing the number of hidden words, until the whole text is memorized.
    To cover the word, click once. To see the word, just move the mouse over it. To mark a word as memorized, click it second time.
